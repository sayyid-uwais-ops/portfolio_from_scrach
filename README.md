# portfolio_from_scrach

💼 Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my profile, skills, and projects.

🌟 Overview

This portfolio represents who I am as a Robotics Engineer & Developer.
It highlights my work, projects, and provides a way to connect with me.

🚀 Features

🎨 Clean and modern UI design

📱 Fully responsive layout (mobile-friendly)

🧭 Multi-page navigation (Home, About, Projects, Contact)

🖼️ Interactive project showcase with hover effects

✨ Smooth UI animations and transitions

📬 Contact form interface

🎯 Simple and user-friendly experience

🛠️ Technologies Used

HTML5 – Page structure

CSS3 – Styling, layout, animations

JavaScript (Vanilla JS) – Interactivity

Font Awesome – Icons
